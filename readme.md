# Intro to Regular Expressions in JavaScript

### Workshop outline
1. Why? What? Where?
2. Basic dictionary. Practice: validators
3. String methods. Practice: replace and parse emails
4. RegExp object. Practice: parsing texts
5. Flags
6. Good and bad practices
